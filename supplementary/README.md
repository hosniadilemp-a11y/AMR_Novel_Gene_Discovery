# Supplementary Materials Curation Folder

This directory contains research tables and datasets supporting the manuscript's findings.

## TSV Data Tables

*   `Table_S1_Assembly_QC.tsv`: Draft genome assembly quality assessment parameters (N50, L50, GC, mosdepth read-mapping depth).
*   `Table_S2_ST354_Cohort_Accessions.tsv`: NCBI GenBank accessions for the 27 clinical cohort genomes and 9 outgroup reference genomes.
*   `Table_S6_Codon_Bias_Statistics.tsv`: Gene statistics (length, GC, GC3 codon bias) and Mann-Whitney U hypothesis testing comparisons across core, accessory, singleton, and acquired resistance gene partitions.
*   `Table_S8_Foldseek_Alignments.tsv`: Reconciliation inclusion matrix detailing length, Pfam domains screen, GC depression significance, and TM-score fit parameters for the 23 candidates.
*   `Table_S11_PanGWAS_Results.tsv`: Top 50 significant orthologous gene hits associated with the ST354 lineage pangenome-wide association studies (PanGWAS).

## Full Datasets

*   `pan_gwas_association_results.csv`: Full list of all 9,184 orthologous gene clusters analyzed by PanGWAS along with sensitivity, specificity, Odds Ratio, Fisher's exact test p-values, and FDR Q-values.
