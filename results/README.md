# Pipeline Analysis Results

This directory contains intermediate and final analysis tables generated by the prioritized candidate identification pipeline.

## Directory Structure

*   `step5_candidates/`
    *   `prioritized_candidates.tsv`: Table listing the 23 novel candidate proteins along with GC% content, lengths, genomic context coordinates, and homology validation fields.
    *   `advanced_stats_results.json`: JSON output containing descriptive stats and hypothesis testing results across genome partitions.
*   `step7_plm/`
    *   `esm2_650m_embeddings.npy`: Numpy binary matrix storing 1280-dimension protein sequence embeddings computed using ESM-2 (650M).
    *   `esm2_650m_coordinates.tsv`: 2D projection coordinates (UMAP and t-SNE) for candidates and reference AMR genes.
    *   `plm_distance_matrix.tsv`: Pairwise Euclidean distances computed in the high-dimensional ESM-2 embedding space.
*   `step7_docking/`
    *   `docking_specificity_results.tsv`: AutoDock Vina binding affinities (kcal/mol) for candidates against standard ligands.
    *   `mmgbsa_binding_energies.tsv`: Prime MM-GBSA binding free energies (kcal/mol) including electrostatics, vdW, and polar solvation terms.
*   `step7_blast/`
    *   `KNGPFPPJ_*_blast_hits.tsv`: NCBI nr BLASTP homolog tables (top 200 hits) for the priority candidate sequences to evaluate taxonomic distribution.
